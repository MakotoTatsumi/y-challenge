## Directory構成

```
├── configs -> 設定ファイル系
│   ├── aspida
│   ├── cspell
│   ├── eslint
│   └── test
├── spec -> OpenAPIから作成したAPI定義
└── src -> 作業ディレクトリ
    ├── api -> 利用するAPI用の関数
    │   ├── modules
    │   └── resas
    ├── components -> atomicデザインに寄せたcomponent構成
    │   ├── atoms
    │   ├── molecules
    │   └── template
    ├── libs -> Apsidaから生成されるファイル 手動では触らない
    │   └── apiClient
    ├── pages
    │   └── modules
    ├── public
    └── shared -> 共通の型定義やutils関数
        ├── types
        └── utils
```
